对比:用 viewport media query 的弊端

/* ❌ 旧方式:看屏幕宽度 */
@media (min-width: 768px) {
  .card { flex-direction: row; }
}
/* 问题:如果同一个 card 在 sidebar(窄)和 main(宽)同时出现,
   一个 viewport 宽度无法兼顾两个位置 */

/* ✅ 新方式:看容器自己宽度 */
@container (min-width: 400px) {
  .card { flex-direction: row; }
}

额外:容器查询单位 cqi / cqb / cqw / cqh

/* 字号根据容器宽度自动调整 */
.title { font-size: clamp(16px, 5cqi, 32px); }
/* 5cqi = 容器内联尺寸的 5% */
