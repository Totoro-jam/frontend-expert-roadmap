IntersectionObserver — 替代 scroll listener

滚动页面,看哪些卡片进入视口(无需注册 scroll 事件,浏览器原生节流)

对比传统方式

// ❌ 老方式
window.addEventListener('scroll', () => {
  cards.forEach(c => {
    const rect = c.getBoundingClientRect()   // 每张卡片都触发 layout!
    if (rect.top < window.innerHeight) load(c)
  })
})

// ✅ 现代
new IntersectionObserver(entries => { ... }).observe(card)

真实场景:

  • 图片懒加载(`loading="lazy"` 之外的精细控制)
  • 无限滚动(观察 sentinel 元素)
  • 曝光埋点统计(广告 / 内容卡片)
  • 视频自动播放/暂停(进出视口)
