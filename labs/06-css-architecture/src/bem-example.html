BEM:Block / Element / Modifier

普通卡片

这是默认状态的卡片。Block 是 card,Element 是 card__title / card__body。

推荐卡片 + 小标题

card--featured 是 Block 修饰符。card__title--small 是 Element 修饰符。

命名约定

  • .block — 独立组件
  • .block__element — 组件内部元素(双下划线)
  • .block--modifier — 状态/变体(双连字符)
  • .block__element--modifier — 元素的状态

BEM 的优势

  • 看到 class 就知道关系(card 是块,card__title 是它的子元素)
  • 选择器永远只有一层(全 class),优先级永远相同
  • 跨框架通用,无需工具链
  • 修改某个组件不影响其他

BEM 的痛点

  • class 名长且重复(card card--featured card--dark)
  • 嵌套层级深时(card__footer__button__icon)很丑 —— BEM 反例,建议改成新 Block
  • 靠纪律,初级开发者很容易破坏约定 —— 用 Stylelint 强制
