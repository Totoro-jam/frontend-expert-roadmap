核心 API

/* 跟随整个文档滚动 */
animation-timeline: scroll();
animation-timeline: scroll(root block);   /* 主滚动条,块方向 */
animation-timeline: scroll(nearest inline); /* 最近滚动祖先,行方向 */

/* 跟随元素进入/离开视口 */
animation-timeline: view();
animation-range: entry 0% cover 50%;
/*
  entry:  元素开始进入视口
  contain: 元素完全在视口内
  cover:  从进入开始到完全离开
  exit:   元素开始离开视口
  例: entry 0% cover 30% = 从一进入到 cover 进度 30%
*/

/* 命名 timeline */
.scroller { scroll-timeline: --my-scroll inline; }
.element  { animation-timeline: --my-scroll; }

/* 命名 view */
.target  { view-timeline: --my-view; }
.linked  { animation-timeline: --my-view; }

真实项目 tips

  • animation-fill-mode: both 让动画状态保留
  • 用 transform / opacity,别动 width/height
  • scroll() 在主滚动条,通常 root 比 nearest 更稳
  • view() 配 animation-range 让动画精准对应位置
  • 嵌套滚动容器:scroll(nearest) 找最近的
  • polyfill:scroll-timeline-polyfill
  • 调试:Chrome DevTools → Animations 面板
  • iOS Safari 17.5+ 支持 view() 但不支持 named scroll-timeline
